Function: 连续截取单屏或者多屏的全屏截图

Parameters: 参数一：Interval截屏间隔时间， 参数二:图像质量Quality (取值：1-100)

Sample_A: 没有参数，间隔一秒钟截取，并且图像质量按PrintScreen 图像质量的50%处理

Sample_B: 接一个参数 [e.g.CaptureScreenShot.exe 3] 间隔三秒截屏一次，图像质量默认按50%处理

Sample_C: 接两个参数 [e.g.CaptureScreenShot.exe 3 100] 间隔三秒钟截屏一次，并且图像质量无损

默认存储路径：运行程序时会显示路径

配置文件路径: D:\HSW_TMS\local_config\CaptureScreen.txt  通过修改盘符可以改变存储路径
